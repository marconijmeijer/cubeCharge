# Homey CubeCharging Plugin

This Homey app allows you to control CubeCharging charge boxes via the official API (https://portal.cubecharging.com/wiki-api/charge-box).

## Features
- Discover and add CubeCharging charge boxes as Homey devices
- Start and stop charging sessions
- View charge box status

## Setup
1. Install this app on your Homey.
2. Pair your CubeCharging account and add your charge boxes.
3. Control charging from Homey flows or the device view.

## Configuration
- You will need your CubeCharging API credentials (see their documentation).

## Development
- Node.js and Homey Apps SDK required
- See Homey developer docs: https://apps.developer.homey.app/

---

This is a starter template. Implement API integration and device logic in `drivers/` and `lib/` folders.
